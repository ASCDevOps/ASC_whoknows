### Playwright tests
#### Initialize Playwright ###

cd into go_app.

```bash
cd go_app
```
then install playwright.

```bash
npm install
npx playwright install
```

#### Use Playwright ###
First cd to go_app.

```bash
cd go_app
```
then run the test with.

```bash
npx playwright test
```
Test results can be viewed in test-results
